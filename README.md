# P. Rahul Reddy — Portfolio

Personal portfolio for P. Rahul Reddy, an Edge AI & Computer Vision Engineer.

Built with React 19, TypeScript, and Tailwind CSS 4 using Vite as the bundler. Emphasizes a high-tech aesthetic and dynamic UI layouts.

## Running Locally

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the local server:
   ```bash
   npm run dev
   ```
3. Open `http://localhost:3000` in your browser.

## Deployment

To generate a static production build:

```bash
npm run build
```

This will create a `dist/` directory, which can be deployed to any static hosting provider (e.g., Vercel, Netlify, or GitHub Pages).
